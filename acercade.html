Acerca de Mí

¡Hola! Soy Emily, una desarrolladora web e Ingeniera en Sistemas apasionada por la tecnología y la creación de soluciones digitales. Me encanta transformar ideas complejas en interfaces de usuario elegantes, funcionales y accesibles.

Mi objetivo es siempre escribir código limpio y mantenible, y me esfuerzo por estar en constante aprendizaje para mantenerme al día con las últimas tecnologías del mundo del desarrollo web.

Mis Habilidades

HTML5 CSS3 JavaScript React Node.js SQL Git y GitHub Diseño Responsivo Figma

Mi Filosofía

Creo firmemente en la colaboración y la comunicación. Disfruto trabajando en equipo para resolver problemas y construir productos de los que todos podamos estar orgullosos. Para mí, un proyecto exitoso no solo funciona bien, sino que también ofrece una experiencia de usuario excepcional.
